Drone Basketball 2D Video Game

Back to Projects

Overview

Designed and tuned a control system for a virtual quadcopter competing in a 1-vs-1 drone-basketball simulation using MATLAB and Simulink. Implemented PID controllers and state-machine logic for motion control, ball handling, and shot execution.

Challenges

  • Tuning multi-axis PID controllers for stable, responsive flight.
  • Applying root locus and pole placement without overshoot.
  • Refining AI-assisted code for consistent simulation behavior.
  • Coordinating ball tracking, positioning, and release timing.

Outcome

  • Achieved smooth, stable control across all axes.
  • Integrated MATLAB code, Simulink blocks, and state logic successfully.
  • Delivered a reliable, autonomous drone player.
